# gdiplusShowcase
Hier finden sich Beispiele zur Arbeit mit GDI+ unter WindowsForms.

Jede Form hat ein eigenes Thema. Es sind immer nur die .cs Dateien (nicht Designer.cs oder .resx) relevant.
